Команды:

Команда: /start_ai - Запустить ИИ
Команда: /stop_ai - Остановить ИИ
Команда: /reset_ais - Сбросить все сессии ИИ

После запуска для создания опроса необходимо соблюсти следующие условия:
1. Бот должен иметь права администратора
2. Тегнуть бота и в сообщение должно быть упоминание о создание опроса
Например: создай опрос про кошек

Команда: /query_poll - Создать опрос
Формат: /query_poll "Название опроса" "Вариант 1" "Вариант 2" "Вариант 2"
Примеры:
/query_poll "Лучший язык?" "Python" "JavaScript" "Rust"

Команда: /stop_poll - Закрыть опрос
Формат: /stop_poll
Требование:
ответить на сообщение с опросом

Команда: /duplicate_poll - Дублировать опрос
Формат: /duplicate_poll (можно добавить "Новое название" "Новый вариант 1" ...)
Примеры:
/duplicate_poll 
/duplicate_poll "Новый опрос" 
/duplicate_poll "Новый" "Вариант 1" "Вариант 2"
Требование:
ответить на сообщение с опросом

Команда: /delete_message - Удаление сообщения
Формат: /delete_message
Требование: 
 1. бот должен иметь права на удаление сообщений 
 2. ответить на сообщение для удаления
